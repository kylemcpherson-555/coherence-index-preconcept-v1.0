## Overview

This repository hosts the **Pre-Concept Working Paper for the Coherence Index**, an early-stage academic framework proposing a new systemic measurement method for assessing coherence across informational, biophysical, human, and economic domains.

This document represents the final non-enabling public release prior to patent submission for the underlying mathematical formulation and source-code architecture of the Coherence Index and the Regenerative Coherence Index (RCI).

The PDF is provided for transparency, timestamped provenance, and academic accessibility.
